## What is Shizuku?

**Shizuku** is a revolutionary Android application that enables regular apps to access system-level APIs with elevated privileges using ADB (Android Debug Bridge) on non-rooted devices. It acts as a bridge between user applications and system services, providing near-root capabilities without the security risks of traditional rooting. This is the definitive way to use **Shizuku Apps** for system-level modifications.

**Key Benefits:**

*   **No Root Required**: Access advanced features safely
*   **Wireless Setup**: Works with Android 11+ wireless debugging
*   **170+ Compatible Apps**: Extensive ecosystem support
*   **Active Development**: Regular updates and community contributions
*   **Universal Compatibility**: Works across all major Android OEMs

## üí° Pro Tips

*   **Wireless Debugging**: On Android 11+, use the built-in wireless debugging feature to start Shizuku without a PC.
*   **Battery Optimization**: To ensure Shizuku and its apps run reliably, disable battery optimization for the Shizuku app.
*   **Permissions**: Always review the permissions requested by any Shizuku-enabled app before granting access.

## üîß Setup Guide

1.  **Install Shizuku**: Download the official Shizuku app from the [Google Play Store](https://play.google.com/store/apps/details?id=moe.shizuku.privileged.api).
2.  **Start the Service**: Follow the in-app instructions to start the Shizuku service via ADB (wired or wireless).
3.  **Grant Permission**: When a compatible app is launched, Shizuku will prompt you to grant it permission to use the Shizuku service.
